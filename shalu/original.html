What is hyperLinks??

=> A hyperlink is a clickable element (text, image, or button) that navigates to another web page, document, or resource.

Created using the <a> (anchor) tag with the href attribute in HTML.

Can link to:

> Another webpage (External Link)

> A section of the same page (Internal Link)

> An email address (Email Link)

> A downloadable file (File Link)

Supports different link behaviors using attributes like target="_blank" (opens in a new tab).

Enhances web navigation and user experience.
